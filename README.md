# HashLips를 이용한 NFT Collection 100개 이상 만들
> node index.js 

NFT Collection 만들기

> npm run update_info 

이미 만들어진 NFT Collection들 업데이트하기

배경 이미지로서 활용할 다양한 파트별 이미지 파일들과 ICON 이미지들을 여러 확률로 랜덤 조합을 통해 NFT Collection으로 사용할 100장 이상의 서로 다른 새로운 이미지 파일들과 JSON 파일들로 빌드받을 수 있다.
